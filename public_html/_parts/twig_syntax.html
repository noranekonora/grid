継承

{% extends '相対パス' %}

インクルード

{% include '相対パス' %}

コメントアウト

{# ここにコメントを書く#}

変数

変数表記

{{ 変数名 }}

変数定義

{% set 変数名 = 'value' %}
  • 配列の場合は
    {% set array = [ A, B ] %}
  • 連想配列は
    {% set array = [ key : value ] %}

条件式

{% if 条件式 %}
  処理
{% endif %}

else文、elseif文、複数条件の場合はandやorも使用可能

{% if 条件式A and 条件式B %}
  処理
{% elseif 条件式 %}
  処理
{% else %}
  処理
{% endif %}

<nav id="ナビ名">
  <ul>
    <li><a{% if currentID |length > 0 and currentID == "1" %} class="current"{% endif %} href="">メニュー1</a></li>
    <li><a{% if currentID |length > 0 and currentID == "2" %} class="current"{% endif %} href="">メニュー2</a></li>
    <li><a{% if currentID |length > 0 and currentID == "3" %} class="current"{% endif %} href="">メニュー3</a></li>
    <li><a{% if currentID |length > 0 and currentID == "4" %} class="current"{% endif %} href="">メニュー4</a></li>
    <li><a{% if currentID |length > 0 and currentID == "5" %} class="current"{% endif %} href="">メニュー5</a></li>
  </ul>
</nav>
  • 「currentID」は任意の名前に変更OK
